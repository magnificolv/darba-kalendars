✍️ MANUĀLĀ IEVADE (bez AI / old-school)

Nekādu mākslīgā integra. Izvēlies mēnesi, un pats ieraksti katru dienu (b / D / tūres nr. / -). Beigās pievieno tūru PDF un veido kalendāru.

Visi lauki sākumā = brīvdiena (b). Klikšķini un pārraksti.
⇄ SINHRONIZĀCIJA PC ↔ TELEFONS

PC un telefons katrs glabā savu localStorage — tie neapvienojas paši. Lai pārceltu mēnesi: Eksportē failu uz vienu pusi → uz otras Importē. (Ja redzi tukšu kalendāru telefonā, visticamāk data.json serverī ir tukšs.)

📋 GRAFIKA PROMPT (universāls — Gemini / Claude / Hermes…)
Lūdzu nolasi šo darba grafika bildi un izvadi tabulu šādā formātā:
1: /70174
2: b
3: 7117
...
Kur "/" pirms numura nozīmē naktsmainis (darbs sākas iepriekšējā dienā),
"b" = brīvdiena, "D" = dežūra 16h (05:00–21:00), "D10" = dežūra 10h.
Ja dežūru seko svītra "-" nākamajā dienā — to arī izvadi kā "-" (nevis tūres nr.).
Tūres numuri ir 4-5 ciparu skaitļi (piem., 5001, 7117, 701/74).
Neraksti neko citu, tikai šo tabulu.
✓ Prompt nokopēts! Ielīmē to savā AI čatā kopā ar grafika bildi.

📸 AUTO OCR (neobligāts)

Nav iebūvētu API noslēpumu — katrs ievada savu API key (Nous / OpenRouter / cits vision-compatible). Worker tikai pārsūta pieprasījumu. Bez key — manuāli / Gemini paste.

🔒 Auto OCR bloķēts, līdz ievadi atslēgu sadaļā ⚙️ OCR iestatījumi zemāk. Bez atslēgas lieto prompta copy-paste (Gemini / Claude / Hermes…).
⚙️ OCR IESTATĪJUMI (advanced / sava atslēga)

Kolēģim: paņem API key no sava Nous / OpenRouter konta → ielīmē šeit → Saglabāt → 📸 bilde. Worker tikai pārsūta pieprasījumu ar viņa key — ne tērē Magnifico kredītus.

🔐 Repo un GitHub Pages nesatur API atslēgas. LocalStorage ir tikai šajā ierīcē/pārlūkā. Rate limit ~20 bildes/stundā/IP. Der jebkurš OpenAI-compatible vision (/chat/completions). Nous: modelis google/gemini-3.5-flash (ne 2.5).
